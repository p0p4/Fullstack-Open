## Usage

### Run Application

```shell
# in project root
npm install
npm run dev -- --host
```

### Run JSON Server

```shell
# in project root
npm run server
```

### Clone the directory

```shell
git clone -n --depth=1 --filter=tree:0 \
  git@github.com:p0p4/Fullstack-Open.git
cd Fullstack-Open
git sparse-checkout set --no-cone part6/redux-anecdotes
git checkout
```
